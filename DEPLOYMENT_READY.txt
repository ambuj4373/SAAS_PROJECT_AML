🔗 INTEGRATION WITH APP.PY

Add to your company check routing:

    from core.french_company_check import run_french_company_check, detect_company_country
    
    # In your company check function:
    country = detect_company_country(company_id)
    
    if country == "France":
        result = run_french_company_check(
            siren=company_id,
            website_url=website,
            tavily_search_fn=your_search_function,
            adverse_search_fn=your_adverse_media_function,
            fatf_screen_fn=screen_entity
        )
    else:
        result = run_company_check(company_number=company_id, ...)
    
    # Existing rendering logic works unchanged!
    render_company_result(result)

The return format is identical for both UK and French checks, so your
existing Streamlit components will work without any modifications.
