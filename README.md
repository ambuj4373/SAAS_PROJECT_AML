# Know Your Charity & Company UK — V4

Intelligence-grade due-diligence screening for UK charities and companies.

## V4 Highlights
- **Bayesian Risk Scoring** — 5-dimensional confidence with diminishing returns & cross-category correlation
- **Fuzzy Entity Matching** — 7-algorithm ensemble (rapidfuzz, Metaphone phonetic, Jaccard)
- **Adverse Media Intelligence** — Source credibility tiers, temporal decay, negation-aware severity
- **Financial Pattern Analysis** — Severity × confidence scoring per pattern with health integration
- **Interactive Plotly Charts** — Financial trends, geographic risk, ownership
- **LangGraph Pipeline** — Structured processing with fallback
- **Modular Architecture** — Separated models, validators, prompts, UI components
- **Structured Logging** — API call metrics and pipeline timing
